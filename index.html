Bulk Create

Upload a CSV to create multiple orders at once

CSV Format:
Order ID,Customer Name,Phone Number,Address,Landmark,Pincode,Delivery Type,Payment Status,Order Status,Product Name,Quantity,Price

One row per product. Repeat order-level fields for each product row. Orders with existing IDs will be skipped.

Bulk Edit

Update existing orders via CSV — only non-empty fields are updated

CSV Format:
Order ID,Customer Name,Phone Number,Address,Landmark,Pincode,Delivery Type,Courier,AWB Number,Payment Status,Order Status

Leave any column empty to keep the existing value. Order ID is required.
